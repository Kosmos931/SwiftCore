## 2) Архитектура и загрузка

### 2.1 Базовые глобалы

- `SC` — основной namespace.
- `SC.util`, `SC.cfg` — вспомогательные таблицы.
- Метатаблицы: `PLAYER`, `ENTITY`, `VECTOR`.

### 2.2 Система инклюдов

Функции загрузки файлов:

- `SC.IncludeSV(path)` — include только на сервере.
- `SC.IncludeCL(path)` — AddCSLuaFile на сервере / include на клиенте.
- `SC.IncludeSH(path)` — AddCSLuaFile + include.
- `SC.include(path)` — авто-выбор по суффиксу `_sv.lua/_cl.lua/_sh.lua`.
- `SC.include_dir(dir, recursive)` — загрузка всей директории.

## 4) Фракции

Файлы: `core/fractions/core/api_sh.lua`, `api_sv.lua` + файлы конкретных фракций.

### 4.1 Декларативный API фракций

#### Главные функции

- `SC.Fractions.Add(name, displayName)` — создать фракцию (builder-объект).
- `fraction:Register()` — зарегистрировать фракцию.
- `SC.Fractions.Get(name)` / `SC.FGet(name)` — получить фракцию.
- `SC.Fractions.SetBaseFraction(name)` / `SC.FBase()` — базовая фракция.
- `SC.Fractions.AddGroup(groupName, ...)` — логическая группа фракций.
- `SC.Fractions.GetGroup(groupName)` — получить группу.

#### Методы фракции (builder)

- `:SetModels(tbl|string)`
- `:SetWeapons(tbl|string)`
- `:SetSpawn(Vector|table)` / `:AddSpawn(...)`
- `:OnSpawn(fn)`
- `:OnDeath(fn)`
- `:Other(fn)`

#### Динамические Set-методы

`Fraction.__index` динамически создает методы `Set*`, например:

- `:SetMaxHealth(150)` → поле `MaxHealth`
- `:SetMaxArmor(50)` → поле `MaxArmor`
- `:SetRunSpeed(500)` → поле `RunSpeed`
- `:SetRegenHP(true, 10)` → поле `RegenHP = {true, 10}`

Это позволяет быстро расширять параметры без переписывания класса.

### 4.2 Серверные операции с фракцией игрока

- `SC.FSetPly(ply, fractionName)` / `frac.SetPlayerFraction(...)`
  - ставит `NWString` фракции,
  - меняет `team`,
  - модель/руки,
  - HP/Armor,
  - скорость/прыжок,
  - выдает лодаут,
  - сохраняет в БД.

- `SC.FSpawn(fraction)` / `frac.GetSpawnPoint(...)`
  - выбирает точку: сначала фракционная, потом `SC.Config.Spawns[map]`.

- `SC.FGive(ply, fraction)` / `frac.GiveLoadout(...)`
  - сбрасывает оружие и выдает базовое + фракционное.

### 4.3 Методы PLAYER

- `PLAYER:SetFraction(fractionName)`
- `PLAYER:GetFraction()`
- `PLAYER:GetFractionName()`
- `PLAYER:GetFractionData()`
- `PLAYER:IsInFraction(name|table)`

### 4.4 Текущие фракции в конфиге

- `citizen` (TEAM_CITIZEN)
- `ccg` (TEAM_CCG)
- `ghoul` (TEAM_GHOUL)

Пример объявления:

```lua
TEAM_CCG = FAdd("ccg", "CCG")
    :SetMaxHealth(150)
    :SetMaxArmor(50)
    :SetDescription("...")
    :Register()
```

## 5) Экономика (Money)

Файлы: `core/systems/money/money_sh.lua`, `money_sv.lua`, `money_cl.lua`.

### API

- `PLAYER:GetMoney()`
- `PLAYER:SetMoney(amount)`
- `PLAYER:AddMoney(amount)`
- `PLAYER:TakeMoney(amount)`
- `PLAYER:HasMoney(amount)`

### Поведение

- Деньги хранятся через `NWInt` ключ `SC_Money`.
- На сервере `SetMoney` синхронизирует значение в `SC.Database`.

### Пример

```lua
if ply:HasMoney(500) then
    ply:TakeMoney(500)
else
    ply:ChatPrint("Недостаточно денег")
end
```

## 6) Голод (Hunger)

Файлы: `core/systems/hunger/*.lua`.

### API

- `PLAYER:GetHunger()`
- `PLAYER:SetHunger(amount)`
- `PLAYER:AddHunger(amount)`
- `PLAYER:TakeHunger(amount)`
- `PLAYER:IsHungry()`
- `PLAYER:GetHungerPercent()`

### Логика

- `SC.Hunger.Max = 100`
- Таймер голода на игрока при `PlayerSpawn`.
- Если голод `<= 0`, наносится периодический урон.

### Пример

```lua
if ply:IsHungry() then
    ply:ChatPrint("Нужно поесть")
end
```

## 7) Стамина (Stamina)

Файлы: `core/systems/stamina/*.lua`.

### API

- `PLAYER:GetStamina()`
- `PLAYER:SetStamina(amount)`
- `PLAYER:AddStamina(amount)`
- `PLAYER:TakeStamina(amount)`
- `PLAYER:GetStaminaPercent()`

### Логика

- Уходит при спринте и прыжке.
- Восстанавливается в движении/в покое/в воздухе по разным коэффициентам.
- При `SetupMove` прыжок блокируется при низкой стамине (`JumpLimit`).

### Пример

```lua
if ply:GetStaminaPercent() < 20 then
    ply:ChatPrint("Ты выдохся")
end
```

## 10) ВСЕ команды (консоль + чат)

## 10.1 Обертки запуска команд

- Консоль: `sc`, `badmin`, `_sc`
- Чат: `/command ...` и `!command ...`

Примеры:

- `sc help`
- `sc setmoney Player 5000`
- `/ban "Player Name" 1h flood`
- `!setrank STEAM_0:1:123456 vip 10mi`

## 10.2 Команды фракций

- `sc_listfractions`
  - выводит список доступных фракций.

## 10.3 Админ-команды (из core модуля)

1. `sc setrank <target> <rank> [time] [exp_rank]`
   - Флаг: `a`
   - Ставит ранг (возможен временный).

2. `sc ban <target> <time> <reason>`
   - Флаг: `b`
   - Бан игрока на указанное время.

3. `sc perma <target> <reason>`
   - Флаг: `b`
   - Перманентный бан.

4. `sc kick <target> <reason>`
   - Флаг: `c`
   - Кик игрока.

5. `sc setmoney <target> <amount>`
   - Флаг: `d`
   - Установить деньги.

6. `sc sethunger <target> <amount>`
   - Флаг: `e`
   - Установить голод.

7. `sc sethealth <target> <amount>`
   - Флаг: `e`
   - Установить здоровье.

8. `sc setarmor <target> <amount>`
   - Флаг: `e`
   - Установить броню.

9. `sc info <target>`
   - Флаг: `f`
   - Информация об игроке.

10. `sc setfraction <target> <fraction>`
    - Флаг: `g`
    - Сменить фракцию.
    - Алиас: `sc setfrac ...`

11. `sc testflags [target]`
    - Флаг: нет
    - Отладка флагов/ранга.

12. `sc help`
    - Флаг: нет
    - Печатает доступные команды.

## 13) Примеры практического использования API

## 13.1 Добавить новую фракцию

```lua
TEAM_MEDIC = SC.FAdd("medic", "Медик")
    :SetMaxHealth(120)
    :SetMaxArmor(25)
    :SetWeapons({"weapon_medkit", "weapon_pistol"})
    :SetWalkSpeed(190)
    :SetRunSpeed(300)
    :OnSpawn(function(ply, fraction)
        ply:ChatPrint("Вы зашли за медика")
    end)
    :Register()
```

## 13.2 Выдать игроку награду

```lua
hook.Add("OnNPCKilled", "RewardMoney", function(npc, attacker)
    if IsValid(attacker) and attacker:IsPlayer() and attacker.AddMoney then
        attacker:AddMoney(150)
    end
end)
```

## 13.3 Создать новую админ-команду

```lua
SC.AdminCommands.Create("freeze", function(ply, args)
    local target = args.target
    if not IsValid(target) then return end
    target:Lock()
    if SC.Admin and SC.Admin.Notify then
        SC.Admin.Notify.AllStaff((ply:Nick() .. " заморозил " .. target:Nick()), ply)
    end
end)
:AddParam("player_entity", "target")
:SetFlag("c")
:SetHelp("Заморозить игрока")
:SetIcon("icon16/lock.png")
```
